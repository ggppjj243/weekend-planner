# 🎯 周末活动规划师

AI智能工作流应用，帮助用户规划完美的周末活动。

## ✨ 功能特点

- **7节点智能工作流**：需求收集 → 天气判断 → 预算评估 → 活动推荐 → 方案对比 → 最终决策 → 详细攻略
- **非线性决策**：根据天气、预算等因素动态调整推荐策略
- **3套活动方案**：AI生成多样化选择，包含对比分析
- **详细攻略**：时间安排、注意事项、携带物品、备选方案

## 🚀 快速启动

### 方式1：一键启动（Windows）
```bash
start.bat
```

### 方式2：手动启动
```bash
npm install
npm start
```

然后访问 http://localhost:8080

## 🔑 API配置

支持以下AI服务商：
- 腾讯混元
- 硅基流动 DeepSeek（推荐，免费额度）
- OpenAI
- 智谱 GLM

## 📝 使用流程

1. 配置API Key
2. 填写出行需求（人数、预算、天气、偏好）
3. 点击"开始智能规划"
4. 查看AI生成的3个方案
5. 查看详细攻略和注意事项
